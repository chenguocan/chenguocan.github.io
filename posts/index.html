搭建自己的Hugo博客 到hugo网站下载响应的安装包。hugo安装包 进入到hugo帮助网页hugo帮助网页 Step1我们已经下载过了,所以我们从Step2开始 Step2 在命令行输入 $ hugo new site 自己的github名字@github.io Step3 在命令打开Step2创建出来的网页,在当前目录的命令行输入 $ git init $ git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 上面成功之后再输入 $ echo 'theme = "ananke"' >> config.toml Step4 在当前目录输入 $ hugo new posts/自己的博客主题.md $ code posts/自己的博客主题.md 注意: 当中如果要让自己的博客不是草稿要让draft:false Step5 开启hugo服务在命令行输入,会生成响应的地址 $ hugo server -D 在config.toml中修改自己的博客信息。执行 $ hugo -D 如果想要在GitHub中上传。要创建.gitignore不让public文件上传到GitHub。在config.toml中修改URL地址为GitHub Pages的地址。 在public目录中。按照GitHub远程仓库上传操作上传
